# 3 Ekranlı Mini Film Uygulaması

Firebase Firestore + Storage ve React Query kullanılarak geliştirilen küçük bir film yönetim uygulaması. Uygulama 3 ekrandan oluşur:

1. **Film Listesi**: Tüm filmleri listeler, detay sayfasına geçiş sağlar.
2. **Film Detayı**: Seçili filmin bilgilerini gösterir; **Düzenle** ve **Sil** işlemleri yapılır.
3. **Film Ekle**: Yeni film ekleme formu.

---

##  Özellikler

* Firestore üzerinde **CRUD** 
* Firebase Storage’a **poster görseli yükleme**

---

##  Teknolojiler

* **ReactNative**
* **Firebase**: Firestore & Storage
* **React Query** 
* **React Router**/**Expo Router**
